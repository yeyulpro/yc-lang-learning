Use the following global design system for ALL generated components.

STYLE:

* Futuristic AI-powered SaaS
* Cyberpunk + Glassmorphism aesthetic
* Clean but immersive UI
* Desktop-first responsive web app

COLOR SYSTEM:

* Background: dark navy/slate gradient
* Surface: semi-transparent glass panels
* Primary accent: cyan/neon blue
* Secondary accent: purple/pink glow
* Text: near-white
* Muted text: slate gray
* Borders: subtle white opacity borders

TYPOGRAPHY:

* Modern sans-serif
* Clear visual hierarchy
* Large bold section titles
* Comfortable body spacing

COMPONENT RULES:

* Use consistent rounded corners (xl/2xl)
* Use soft glow and backdrop blur
* Avoid random colors
* Avoid flat white cards
* Inputs and buttons must visually match
* Consistent padding and spacing

LAYOUT RULES:

* Components must fit naturally inside parent layouts
* Avoid oversized cards
* Maintain visual balance
* Use shared spacing rhythm

ANIMATION:

* Smooth transitions only
* Subtle hover glow
* Avoid excessive motion

IMPORTANT:

* Match ALL existing components visually.
* Do not invent a new style direction.
* Maintain one unified design language.
